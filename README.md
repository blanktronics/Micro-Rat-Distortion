# Overview
Micro Rat Distortion majorly influnced by Olinthus Designs mini effects.

The idea is simple, I wanted to challenge myself by trying to recreate an extremely small version of the ProCo Rat distortion.

My secondary challenge is I want to create something even smaller than the Olinthus Desgin effects.

So far the dimesions of my project mostly mimcs the Olinthus dimensions except my assembly is 50mils wider and 50 mils thinner.

Next steps are to just shrink shrink shrink.

If you would like to try this out for yourself, the KiCad project I've designed can found in this repository!
